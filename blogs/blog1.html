Spring Break RPG Prototype

Given one week of freedom from the grueling hell of being a computer engineering student, I began working on a prototype for an rpg game that I wanted to create.

I first implemented the player's movement ability. I decided to go for a grid based movement system to create a JRPG feel to my game.

Once that was done, I gave the player NPCs to interact with and created a dialogue system

Now that the dialogue system has been established, I began working on the game menu. The menu contains four options: Character,Items,Exit, and Save

A JRPG would not be a JRPG without its turned based combat to which I modeled the characters after Final Fantasy 7 characters

And finally, I created a save and load system for the game to save the player's progress.

The sprites I used to prototype this RPG is from an opengameart.org user named Sharm

This prototype is still no where close to being finished. I still need to implement a questing system and ways to create a story within the game. However, I made substaintial progress during spring break and hopefully, I can finish this prototype before the start of summer break
